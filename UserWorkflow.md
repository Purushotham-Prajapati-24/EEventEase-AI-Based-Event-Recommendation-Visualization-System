. Customer Journey Mapping
🧑‍🎓 Persona: Student (Primary User)
📍 Journey Stages
1. Awareness
Learns about platform via college or peers

Pain Point: Too many scattered event sources

2. Onboarding
Visits platform
Selects interests

Emotion: Curious
Goal: Get relevant events

3. Discovery
System shows top recommendations

Experience:

Clear UI
Personalized results

Pain Point Solved: No need to search manually

4. Understanding
Views:
Match score
Popularity
Explanation

Emotion: Trust

5. Decision
Checks:
Timeline
Trends
Decides which event to attend
6. Action
Registers for event
7. Post-Experience
Rates event
Gets better recommendations next time
🔁 Continuous Loop
More usage → better recommendations → higher engagement
